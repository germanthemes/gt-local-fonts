== Description ==

The *GT Local Fonts* plugin adds additional local fonts to the typography options of German Themes.

Loading external Google Fonts is problematic with the GDPR and European privacy laws. This plugin includes the most popular Google Fonts and allows them to be embedded locally. It works with all themes from GermanThemes, or any theme which adds support for the plugin.

== Installation ==

1. Activate the plugin
2. Go to Customize > Theme Options > Typography to use the new custom fonts.
